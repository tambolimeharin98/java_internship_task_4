# Collections Framework & Generics Task

## Description
This project demonstrates the use of Java Collections Framework and Generics.

## Features Implemented
- List (ArrayList) to store employees
- Set (HashSet) to maintain unique employees
- Map (HashMap) for ID-based lookup
- Generic class DataSorter<T> for sorting
- Comparable interface implementation

## Concepts Covered
- Collections Framework
- Generics
- Sorting using Collections.sort()
- equals() and hashCode()
- O(1) lookup using HashMap

## How to Run
Compile and run using:

javac CollectionsTask.java
java CollectionsTask
